Create a file called stack.py that implements a Stack class with the following API:
- push(item): add item to top
- pop(): remove and return top item, raise IndexError if empty
- peek(): return top item without removing
- size() / __len__: return number of items
- is_empty(): return True if empty
- __iter__: iterate from top to bottom (LIFO order)
- __str__: return a human-readable representation

A test file test_stack.py already exists with the expected behavior. Read it first, then implement stack.py. After implementing, run the tests to verify they all pass.
